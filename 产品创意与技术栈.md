# 产品创意与技术栈汇总

> 10 个高价值产品创意及对应技术选型，聚焦可落地、有影响力的方向。

---

## 一、创意列表
### 3. 会议 / 视频通话的实时翻译字幕
- **范围**：为 Zoom、Teams 等会议提供实时多语言字幕，保留语气和停顿。
- **价值**：让跨国会议真正无障碍，降低语言门槛。

### 5. 按掌握程度而非年龄的教育系统
- **范围**：AI 根据个人节奏、风格和薄弱点调整教学，以「掌握」而非「年级」为进度单位。
- **价值**：普及优质教育，重新定义「学校」。

### 8. 去中心化的个人记忆与身份层
- **范围**：用户拥有并控制自己的身份、凭证和「记忆库」，可跨平台、跨时间迁移。
- **价值**：从「平台拥有用户数据」变为「用户拥有自己」，改变权力结构。

### 9. 创作灵感与素材库
- **范围**：随时记录灵感、截图、链接，支持标签、关联和搜索，适合写作者、设计师等。
- **价值**：把零散灵感结构化，方便后续创作和复盘。

### 10. 家庭口述历史 / 祖辈问答录
- **范围**：引导长辈录制口述，子女可提问，系统整理成可检索的「问答录」。
- **价值**：把家庭智慧结构化、可传承，而不是只留下零散视频。

## 二、重点创意技术栈（3、5、8、9、10）

### 3. 会议 / 视频通话的实时翻译字幕

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **语音转文字 (STT)** | Whisper / Azure Speech / 讯飞 | Whisper 可本地部署，云服务延迟更低 |
| **实时翻译** | DeepL API / Google Translate API | DeepL 质量好，Google 语种多 |
| **音频捕获** | WebRTC / 系统音频虚拟设备 | 浏览器扩展用 WebRTC，桌面端可用虚拟声卡 |
| **流式传输** | WebSocket / gRPC | 低延迟流式传输 |
| **前端** | React + TypeScript | 字幕悬浮层、设置面板 |
| **形态** | 浏览器扩展 / Electron 桌面应用 | 扩展覆盖 Web 会议，Electron 可支持桌面客户端 |
| **后端** | Node.js / Python (FastAPI) | 做代理转发、计费、多租户等 |

**架构思路**：浏览器扩展 → 捕获 Tab 音频 → 后端 STT+翻译 → WebSocket 回传字幕。

### 5. 按掌握程度而非年龄的教育系统

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端** | Next.js / React Native | Web + 移动端 |
| **后端** | Node.js (Nest.js) / Python (FastAPI) | 业务逻辑、API |
| **数据库** | PostgreSQL | 用户、课程、知识点、掌握度 |
| **知识图谱** | Neo4j / PostgreSQL + 自建关系 | 知识点依赖、前置关系 |
| **推荐 / 路径** | 规则引擎 + 简单 ML | 根据掌握度推荐下一学习内容 |
| **内容存储** | 云存储 (S3/OSS) + CDN | 视频、文档、习题 |
| **视频** | 自建 / 七牛 / 阿里云点播 | 支持进度、倍速、打点 |
| **评估** | 自研题库 + 间隔重复算法 | 掌握度评估、复习提醒 |

**架构思路**：知识点树 + 掌握度模型 → 推荐引擎 → 学习路径 + 内容推送。

### 8. 去中心化的个人记忆与身份层

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **身份** | W3C DID / Ceramic / Ethereum ENS | 自管身份、可解析 |
| **凭证** | Verifiable Credentials (VC) | 学历、技能等可验证声明 |
| **存储** | IPFS / Arweave / 用户自托管 | 内容去中心化、可持久 |
| **前端** | React + wagmi / ethers.js | 钱包连接、DID 展示 |
| **钱包** | MetaMask / WalletConnect | 登录、签名 |
| **索引** | The Graph / 自建索引服务 | 链上 / 存储 数据查询 |
| **加密** | 用户端加密 (libsodium) | 端到端加密，平台不解密 |

**架构思路**：DID 作为身份根 → VC 存链上或 IPFS → 记忆/数据加密存 IPFS，密钥用户自管。

### 9. 创作灵感与素材库

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端** | Next.js + PWA | Web 优先，可离线 |
| **浏览器扩展** | Manifest V3 | 截图、保存链接、选中文本 |
| **后端** | Node.js / Python | API、任务队列 |
| **数据库** | PostgreSQL | 用户、素材、标签、关系 |
| **向量搜索** | pgvector / Pinecone / Milvus | 语义检索、相似素材 |
| **全文搜索** | PostgreSQL FTS / Meilisearch | 关键词检索 |
| **存储** | S3 / Supabase Storage | 图片、文件、截图 |
| **AI** | OpenAI Embeddings / 本地 sentence-transformers | 文本/图片向量化 |
| **OCR** | Tesseract / 云 OCR | 图片内文字提取 |

**架构思路**：多端采集 → 统一入库 + 向量化 → 标签 + 语义检索 → 关联与推荐。

### 10. 家庭口述历史 / 祖辈问答录

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **录音** | Web Audio API / 原生录音 | 浏览器或 App 内录制 |
| **转写** | Whisper (本地/API) / 阿里/讯飞 | 语音转文字 |
| **问答 (RAG)** | LangChain + OpenAI / Claude / 本地 Ollama | 基于口述内容回答 |
| **向量库** | Chroma / pgvector / Pinecone | 口述内容向量化 |
| **后端** | Python (FastAPI) | 转写、RAG、API |
| **前端** | Next.js / React Native | 录制、提问、浏览 |
| **存储** | 云存储 + PostgreSQL | 音频、转写、元数据 |
| **引导** | 预设问题模板 | 引导长辈回忆、补充话题 |

**架构思路**：录音 → 转写 → 分块 + 向量化 → 用户提问 → RAG 检索 → 生成回答。
