# 开启真实语音识别

当前默认是 **Mock 模式**，只显示测试文字，不会识别你说的话。

要显示**真实说话内容**，需要安装 Whisper 并切换配置。

---

## 步骤

### 1. 安装 Whisper

在 `backend` 目录下执行：

```bash
pip install openai-whisper
```

首次运行会自动下载模型（base 约 140MB），需要一点时间。

### 2. 设置环境变量

**方式 A：命令行**

启动后端时加上：

```bash
set STT_PROVIDER=whisper
python run.py
```

**方式 B：创建 .env 文件**

在 `backend` 目录创建 `.env`，内容：

```
STT_PROVIDER=whisper
```

### 3. 重启后端

关闭当前后端（Ctrl+C），再执行：

```bash
python run.py
```

---

## 验证

启动后，在会议中说话，字幕应显示你说的话（或翻译后的内容）。

若仍无内容，请确认：
- 选择的是「整个屏幕」或包含会议声音的窗口
- 会议中有人在说话
- 麦克风/系统音量未被静音
